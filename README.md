# Progressive_Budget_Tracker

Created an application where, a user will be able to  track withdrawals and deposits with or without a data/internet connection.
So that account balance is accurate when user is traveling          

 ## Criteria


* GIVEN a user is on Budget App without an internet connection
* WHEN the user inputs a withdrawal or deposit
* THEN that will be shown on the page, and added to their transaction history when their connection is back online.
